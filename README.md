## TrackApps Telegram Bot

Бот для отслеживания доступности приложений Google Play. Сохраняет ссылки в MongoDB, каждые 10 минут проверяет доступность и позволяет вручную запустить проверку.

### Запуск
1. Установите зависимости (без виртуалки):
```bash
python3 -m pip install -r requirements.txt
```

2. Налаштуйте змінні середовища. Скопіюйте `.env.example` у `.env` і заповніть значення:
```bash
cp .env.example .env
```
У `.env` обовʼязково вкажіть:
- `BOT_TOKEN` — токен бота
- `MONGO_URI` — рядок підключення до MongoDB

Інші змінні (наприклад `DB_NAME`, `CHECK_INTERVAL_MINUTES`) мають значення за замовчуванням у `.env.example`.

3. Запустите бота:
```bash
python3 main.py
```

### Команды бота
- `/start` — приветствие и клавиатура
- `/add <google_play_link>` — добавить приложение
- `/remove <google_play_link>` — удалить приложение
- `/list` — вывести список приложений и их статусы
- `/check` — вручную проверить все приложения сейчас

### Структура БД
Коллекция `apps`:
- `link`: строка, нормализованная ссылка на Google Play (уникальная)
- `status`: `available` | `unavailable`
- `banned_at`: datetime | null — время, когда приложение впервые обнаружено недоступным
- `renew_at`: datetime | null — время, когда приложение снова стало доступным после бана
- `last_checked_at`: datetime — время последней проверки
- `created_at`: datetime — время добавления записи

### Примечания
- Проверка выполняется каждые 10 минут автоматически.
- Кнопка «Проверить приложения» в клавиатуре запускает проверку вручную.
